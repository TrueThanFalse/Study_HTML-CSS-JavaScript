Javascript의 배열은 유동배열 : 크기,데이터 변경이 가능

  • const myArray = [1,2,'a',4,"b",`c`, ...];
    => 자료형에 크게 구애 받지 않고 저장할 수 있음
    => Java의 ArrayList와 비슷함
  • 하지만 배열의 데이터 타입은 같은 자료형으로 만드는 것이 룰
  • myArray = []; : 초기값이 없는 빈 배열을 만든 후
    데이터를 추가하여 사용하는 것이 일반적임
  • index 가지고 있음 : 0부터 시작
  • length : 배열의 총 길이
  • for of : 향상된 for문 사용 가능 for in : 오브젝트에 사용 가능

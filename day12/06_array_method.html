배열 관련 메서드 : 유사 배열에서는 사용제한이 되는 메서드도 있음

  • array.push(value) : 배열의 마지막에 value를 삽입, 배열의 길이 값을 return
    => array[array.length] = value; 와 같은 의미
  • array.pop() : 배열의 마지막 요소를 제거, 제거된 요소를 return
  • array.shift() : 배열의 첫 요소를 제거, 제거된 요소를 return
  • array.unshift(value) : 배열의 0번지에 value를 삽입, 배열 길이를 return
  • array.join('value') : 배열의 원소 사이에 문자를 삽입하여 새로운 문자열 생성, return이 문자열

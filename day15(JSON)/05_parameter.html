자바스크립트의 파라미터 특징 & 기능

  • 파라미터의 개수와 관계없이 함수나 메서드 이름이 같다면 호출 가능
  • 파라미터의 개수가 불일치하여 값을 받지 못하는 파라미터가 있다면 undefined 처리 함
  • 값을 받아오지 못하는 파라미터가 undefined 처리 되는걸 막기 위해 기본값을 설정할 수 있음 => optional parameter
  • 모든 파라미터를 관리하는 arguments 라는 프로퍼티를 이용하여 파라미터를 관리 함 arguments 배열로 담겨서 관리 됨 (=> 그대로 사용할 수 있음)
  • optional parameter는 arguments에 저장되지 않음
  • optional parameter는 항상 다른 파라미터보다 뒤에 있어야 함(가장 끝에 존재 가능)
  • rest parameter : 남는(이외의, 남는것들) 파라미터 => 사용할 파라미터는 일반 파라미터로 배정하고 나머지 파라미터들을 통합하여 하나의 이름으로 배정하는 기법
  • ... : spread 연산자라고 하고 배열, 객체를 복제하는데 사용 함

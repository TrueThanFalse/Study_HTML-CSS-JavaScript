Map => key / value 형태의 데이터를 한개의 배열로 만들어서 다시 배열에 저장함

  • 선언 및 초기화 : new Map(), new Map([['key','value'],['key','value'],...])
  • 기본적으로 중첩 구조를 가지고 있기 때문에 key 중복 확인이 필수임
  • key를 식별자로 활용하므로 get(keyName), set(keyName, value) 메서드를 사용 가능
  • 2차원 배열 형태와 비슷한 구조임
  • key 타입은 문자로 하는 것이 원칙(String) => key 타입이 여러개면 관리가 어려워지기 때문
  • get(keyName) : key에 해당되는 value 값을 return
  • set(keyName, value) : key에 해당하는 value를 변경 / add 따로 없이 set으로 데이터를 추가함
  • delete(key), has(key), forEach(), keys(), entries(), values(), clear(), size 모두 가능

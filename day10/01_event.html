이벤트 객체는 전역객체로써 모든 태그에 객체를 참조 시킬 수 있음
전역객체 : 모든곳에 영향을 줄 수 있음

  • a 태그와 button 태그에 이벤트를 참조시켜
    클릭에 대한 반응을 기대할 수 있도록 만드는 것이 일반적임
  • a 태그에서는 href 속성에 의해 자체 이동(동기화 방식)이 구현되어있어
    별도의 이동을 막는 코드가 필요 => (href:"#" -> #으로 이동을 막음)
  • 객체에 이벤트를 부여하는 방식에는 크게 3가지가 있음 (각자의 목적과 작동방식이 다름) 1. document.addEventListener 2. innerText 3. innerHTML
  • 화면의 출력 영역을 document(하나의 객체)영역 이라고 함
  • html 태그: html element, 속성: attribute, 일반 텍스트:textNode 라고 함
  • html로 출력하기 : innerHTML 라는 property를 사용
  • text로 출력하기 : innerText 라는 property를 사용
  • html element를 id 값으로 객체 가져오기 : getElementById(`idName`)
    => 주로 ID를 통해 객체를 가져옴
  • html element를 css 선택자를 사용하여 가져오기 : querySelector('selector')

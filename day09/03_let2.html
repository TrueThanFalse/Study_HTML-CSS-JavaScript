JavaScript의 변수 사용

  • 값을 담고 있는 메모리 공간의 이름
  • 자바스크립트에서 변수는 선언하는 역할만 함
  • var, let(ES6 부터는 let을 사용하는 것을 적극! 권장!), const
    var는 중복 선언이 가능하여 무결성을 유지하기가 어려움
    let은 중복 선언 불가
  • Java와 달리 변수에 자료형을 표시하지 않음
    즉, Java처럼 int, String, char 등의 자료형이 존재하지 않음
  • 따라서 let으로 선언하면 어떤 값이든 저장할 수 있음
  • var name = 'hong';
    var name = 'kim'; => 중복 선언 가능
  • let name = 'hong';
    name = 'kim'; => 재할당 가능
    let name 'kim'; => 중복 선언은 불가능
  • 스코프(scope : 사용범위) { }에서만 사용 가능
    해당 스코프를 벗어나면(사라지면) 재선언 가능
    => Java의 지역변수와 같은 개념
  • const : 재선언 불가능, 재할당 불가능
    const name = 'hong';
    name = 'kim'; => 불가능
    const name = 'kim'; => 불가능
  • const는 선언 이후 항상 고정값을 가지는 경우에 사용
    고정된 값을 사용하는 함수, 규칙, 참조, 룰, 객체, 엘리먼트 등을 정의시 사용

함수(function) : 특정 기능을 수행하기 위해 작성한 코드 => 호출되어야만 실행됨
Java 에서는 클래스 안에서 메서드가 생성됨
하지만 JS 는 어디서든 function 생성 가능

  • function 이라는 키워드를 사용하여 정의
  • 작성 구문. function 함수명(파라미터){ 로직 작성; }
  • 함수명 : 기능을 함축하는 의미로 작성
  • 파라미터(매개변수) : 선택적으로 사용 (없어도 ()는 작성해야함)
  • JS 에서 파라미터는 타입(자료형)이 자유도가 높아 다양한 형태로 사용 가능
  • return : 함수 종료 역할과 동시에 특정 결과 값을 호출한 위치로 반환 (Java와 동일)

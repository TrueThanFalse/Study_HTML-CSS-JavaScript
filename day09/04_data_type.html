변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(not a Number : 숫자가 아님)
  • 문자 : String => "" or '' or `` 큰따옴표or작음따옴표or백틱으로 감싼 값
  • 불리언 : boolean => true, false의 형태
  • 객체 : Object => 1개 이상의 데이터가 특정한 구조를 가지고 있는 형태
    e.g.) {키 : 값} ({ } : 아무것도 없어도 1개의 객체로 취급함)
  • 배열 : Array => [1,2,3,...] 형태의 데이터 집합체
    순서, 길이가 있음 -> Java의 List와 비슷한 구조
  • null : 알 수 없는 값 => 뭔가 있는것 같긴한데 파악이 안됨
  • indefined : 미지정된 값 => 값이 할당되지 않음
  • 데이터의 타입을 확인하는 명령어 : typeof 변수명

변수의 관리

  1. 기본자료형 : 값의 메모리의 주소값이 가르키는 곳에 직접 할당되는 값
    number, boolean, String(참조자료형이지만 사용편의상 기본자료형과 같이 사용 가능)
  2. 참조자료형 : 값이 저장되어 있는 주소를 할당
    String, Object, Array, function, null, undefined
  3. == 같다, != 같지않다 : 값만 비교 === 같다, !== 같지않다 : 값과 타입이 같은지 비교

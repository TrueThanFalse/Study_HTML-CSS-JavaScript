비표준 속성(non-standard Attribute)

  • html 태그의 속성이 제한적이므로 데이터를 관리하는 차원에서 개발자들에게 직접 속성을 만들어 사용할 수 있는 방법을 제공함
  • MarkUp 언어들은 사용자(개발자)들에게 표준이 아닌 사용자만의 문법을 사용할 수 있도록 설계되어 있음
  • 너무 과도하게 사용하면 개발 효율(협업)과 가독성 저하 발생
  • 태그 속성의 이름은 data-변수명="value" 하이푼(-) 뒤에 변수명을 사용하여 value 값을 인식할 수 있도록 설계됨
  • dataset.변수명 : 값이 return 됨 dataset property를 사용하여 읽어 들임
  • 변수명에 대문자는 허용X : 대문자를 사용하더라도 소문자로 변환해서 읽음

가장 많이 다루는 데이터 타입 : String

  • Java 에서의 String 메서드와 거의 유사함
  • 문자열 데이터는 index(0부터 시작)를 가지고 있음
  • 문자열 값 내에서 개행(br)하는 것을 추천하지 않음
  • new String(), 객체 타입으로 선언 가능하지만 추천하지 않음
  • length : 문자열의 총 길이
  • 추출 관련 메서드
    ● substring(startIndex, endIndex) : endIndex는 포함X
    ● slice(startIndex, endIndex) : endIndex는 포함X
    ● substr(startIndex, count) : count => 개수
  • 문자 대체 관련 메서드 : replace(old, new) : 첫 매징되는 문자만 가능 (대소문자 구분함)
  • trim() : 양쪽 공백 제거
  • padStart(원하는 문자열 총 길이, 앞쪽 빈공간에 채울 문자) /
    padEnd(원하는 문자열 총 길이, 뒤쪽 빈공간에 채울 문자)
    str1 = '시'; / str2 = '010101-4';
    ****시 => str1.padStart(5,`*`); / 010101-4****** => str2.padEnd(14,`*`);
  • charAt(index) => return이 String임, indexOf() 가능
  • startWith('str') / endWith('str')
    해당 문자열 앞/뒤에 str이 존재하는지 확인하는 메서드
    return이 true / false 로 리턴됨

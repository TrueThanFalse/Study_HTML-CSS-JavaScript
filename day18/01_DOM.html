DOM(Document Object Model) : html(문서) 객체 모델

  • html 하위 태그는 각각의 기능(method), 속성(properties)를 갖고 있는 문서 객체이다.
  • element 가져오기 : tag element를 가져오는 메서드 : byID, byClassName, byTagName 등등 있음
  • document.getElementById('idName') : 단일 element 리턴
  • document.getElementByClassName('idName') : 복수 element 리턴 => HTMLCollection 타입으로 리턴 함
  • document.getElementByClassName('idName') : 복수 element 리턴 => HTMLCollection 타입으로 리턴 함
  • 복수 엘리먼트로 가져오게 되면 index를 선택해야 사용할 수 있음

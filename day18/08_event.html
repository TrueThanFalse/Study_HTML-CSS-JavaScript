document 영역에서 사용되는 이벤트 정리

  • 마우스 이벤트 : click(), focus(), input(), change(), mouseover() mouseout(), scroll() 등등...
  • 키보드 이벤트 : keyDown(), keyPress(), keyUp()
  • 기타 이벤트 : preventDefault(), target, submit()
  • addEventListener('eventName', callback function); ● 메서드를 사용하여 구현하는 것을 권장 ● 태그에 onclick을 적용하는 것보다 addEventListener를 권장함 ● 이유 : 이벤트 중복 부여 가능

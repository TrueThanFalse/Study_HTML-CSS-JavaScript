element의 속성에는 크게 두가지로 구분 ● 태그의 속성 ● CSS(style)의 속성

  • 태그의 속성은 태그가 갖는 속성이기 때문에 속성명을 그대로 사용 tagName.id, tagName.value, tagName.src, tagName.href 등등...
  • class처럼 여러개의 값을 갖는 속성들은 별도의 속성명을 사용 tagName.classList (class만 classList(배열)이다.)
  • CSS 관련 속성들은 style 이라는 속성을 통해 프로퍼티에 접근 가능 tagName.style.color
  • 단, style 속성 중 font-size 처럼 -가 있다면 -를 지우고 첫글자를 대문자로 변경 font-size => fontSize
